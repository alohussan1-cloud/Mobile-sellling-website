Mobile Store Website

A full-stack Mobile Store Website developed using PHP, MySQL, HTML, CSS, and JavaScript as a final course project.

Features

User Side

- Browse mobile phones
- View product details
- Responsive design
- Mobile brand categorization

Admin Side

- Secure admin login
- Protected dashboard routes
- Add new products
- Edit existing products
- Delete products
- Product image uploads
- Product management through dashboard

Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

Database

The project uses MySQL for storing:

- Products
- Admin accounts
- Users
- Orders

Installation

1. Clone the repository.
2. Copy the project folder into the XAMPP "htdocs" directory.
3. Start Apache and MySQL from XAMPP.
4. Create a database in phpMyAdmin.
5. Import the provided SQL file.
6. Update database credentials in the configuration file.
7. Open the project in your browser.
